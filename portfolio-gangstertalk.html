구현 기술

- MERN(Mongo, Express, React, Node) 기반 -
1. 회원가입 중복방지, 암호화(bcrypt), 로그인 시 세션관리, Ajax
2. 검색 결과 데이터 요청 및 출력 : jQuery Ajax JSON axios
3. socket.io를 활용한 실시간 채팅 구현
4. WebRTC를 활용한 p2p 채팅 및 화상 채팅 구현
5. Cloudinary로 포스팅 이미지 저장
6. 회원 검색 가능으로 팔로우 찾기 가능
